# 局部微气候加密专员 (Crypto Micro Agent)

本技能属于“气象观测站 1.0”的同化层（数据融合）Agent 之一。
核心职责是监测流入加密市场的“水”的量、质与流速。

## 核心算法与法则
本专员遵循 L3 认知定海神针中的绝对算法：
1. **稳定币流速**：资金总量大不代表牛市，如果资金沉淀在借贷协议吃息，则属于观望避险。只有活跃度飙升才是真实的看涨。
2. **ETF vs CME**：如果散户主导的现货 ETF 狂热流入，但华尔街机构主导的 CME 期货反应平平，说明缺乏长线“暖流”，反弹极不坚固。

## 工作流与执行指令 (Execution)

当用户请求分析加密市场资金流向、稳定币总市值或机构散户情绪背离时，执行以下步骤：

1. **执行抓取脚本**：
   运行捆绑的 Python 脚本，以拉取最新的 DefiLlama 稳定币和 Yahoo Finance 市场数据。
   ```bash
   python3 scripts/fetch_crypto_micro.py
   ```
2. **分析结果**：
   该脚本会将最新的市值规模及交叉对比情况存入 `L1 Inbox`（`./digital_twin/Inbox/`）。读取最新生成的报告。
3. **输出交叉验证报告**：
   提取“全网稳定币总市值”及“ETF 与 CME 的对比状态”，并结合当前市场语境，推演当前资金的真实意图。
