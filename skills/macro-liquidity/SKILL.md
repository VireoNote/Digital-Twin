# 宏观流动性特征提取器 (Liquidity Plumbing Agent)

本技能属于“交易架构 1.0”的特征提取模块 之一。
核心职责是抓取并计算系统底层流动性的健康状况，并输出雷暴预警。

## 核心算法
本专员遵循 L3 系统约束中的绝对算法：
`Net_Liquidity = WALCL (美联储总资产) - TGA (财政部一般账户) - RRP (隔夜逆回购)`

## 工作流与执行指令 (Execution)

当用户请求分析流动性或你认为当前任务需要调用宏观流动性数据进行交叉验证时，执行以下步骤：

1. **执行抓取脚本**：
   运行捆绑的 Python 脚本，以拉取最新的 FRED 数据并进行计算。
   ```bash
   python3 scripts/calc_net_liquidity.py
   ```
2. **分析结果**：
   该脚本会自动将计算结果及雷暴预警判定存入 `L1 原始事件层`（`./digital_twin/Inbox/`）。读取脚本输出或最新的生成的报告文件，提取“净流动性”数值和“雷暴预警状态”。
3. **输出交叉验证报告**：
   将这部分宏观“气压”数据作为判断依据输出给用户。注意：如果脚本输出【高危】预警（RRP < 200B），必须在你的回答中进行醒目的风险提示。
