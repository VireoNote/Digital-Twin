# 多普勒雷达专员 (Doppler Radar Agent)

本技能属于“气象观测站 2.0”的同化层（数据融合）Agent 之一。
核心职责是监控真金白银计价的预测市场，弥补传统宏观/微观量化模型对于“未来预期”和“突发事件”的盲区。

## 核心算法与法则
本专员遵循 L3 认知定海神针中的绝对算法：
1. **硬性物理过滤**：只抓取大资金（>100万）、强动量、近端（<30天）的事件，忽略长尾死水。
2. **贝叶斯偏移 ($\Delta E_{narrative}$)**：输出的不是绝对利多或利空信号，而是作为环境系数的“偏置项”。

## 工作流与执行指令 (Execution)

当用户请求分析市场预期、突发事件影响或执行 v2.0 完整预报时，执行以下步骤：

1. **执行抓取脚本**：
   运行捆绑的 Python 脚本，以拉取最新的 Polymarket 高动量事件。
   ```bash
   python3 scripts/fetch_doppler_radar.py
   ```
2. **分析结果**：
   该脚本会将过滤后的关键预期事件存入 `L1 Inbox`（`./digital_twin/Inbox/`）。读取最新生成的雷达报告。
3. **输出交叉验证报告**：
   提取当前的高能叙事事件。**作为主控中心，你必须主观判断这些事件的冲击方向（I 值正负）**，并在最终预报中说明它们是如何修正原有宏观底色 $E_{macro}$，从而影响最终微观阈值的。
